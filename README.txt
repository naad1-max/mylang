MATHLANG
====
--> Welcome
* Welcome to my GitHub repository!
* This repository is a math language with variables.
----
--> About
* I created this repository to create my own programming language.
* It ended up being my own math implementation, so I gave up and pushed it to GitHub.
----
--> Usage
* Run the shell: "python3 repl.py".
* Run a file: "python3 run_from_command_line.py path/to/file.mlg" ("mlg" is the mile extension for "mylang").
* Create a file:-
"12 + 3.5
2 * 7.8
10 / 2
7.2 - 5.3

let a = 10
let b = 2
a / b
a * b
a - b
a + b"
* Run it using the "run_from_command_line.py" file.
