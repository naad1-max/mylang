# MyLang
## Features
* File extension `.mlg`
* Basic CLI (`python3 run_from_command_line.py examples/hello.mlg`)
* Interactive REPL shell
* Simple math interpreter

## Usage
Run a file:
```bash
python3 run_from_command_line.py path/to/file.mlg
```
Start REPL:
```bash
python3 repl.py
```

## Examples
```mlg
12 + 3.5
13 - 5
7 * 7
10 / 2
(12 - 6) * 4 + 6 / 5
```
